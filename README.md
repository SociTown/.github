The files in this repository are applied as defaults to repositories in the SociTown organization. Use this repository to maintain shared templates, policies, and automation that should be consistent across projects.

How to update

2. Modify or add files in this repository.
3. Open a pull request against `main` with a clear description of the change and the motivation.
4. Once approved and merged, the templates and workflows will apply to organization repositories according to GitHub's precedence rules.

Contributing

Contributions are welcome. To propose a change:
- Branch from `main` (or fork if you don't have direct push access).

- Open a pull request and request review from the maintainers team.

If the change affects legal or behavioural documents (e.g. CODE_OF_CONDUCT), ensure maintainers' sign-off before merging.
